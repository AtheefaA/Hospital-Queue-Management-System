# 🏥 Smart Hospital Queue Management System

A modern full-stack hospital queue and appointment management system built with **React, Node.js, Express.js, and MongoDB**.

## ✨ Features

* 🔐 User Registration & Login
* 👤 Patient Dashboard
* 👨‍⚕️ Doctor Dashboard
* 🛡️ Admin Dashboard
* 📅 Book Appointments
* 🎫 Live Queue Status
* 📋 My Appointments
* 👨‍⚕️ Doctors Directory
* 👤 User Profile
* 🔑 Secure Password Hashing
* 🌐 MongoDB Database Integration
* 📱 Responsive UI

## 🛠️ Tech Stack

### Frontend

* React
* React Router
* Axios
* CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcryptjs

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm start
```

## 🔑 Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
